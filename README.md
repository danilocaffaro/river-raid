# River Raid - Atari 2600 Clone

A pixel-perfect browser-playable clone of the classic Atari 2600 game River Raid.

## Play Now
Open `index.html` in your browser or visit the GitHub Pages URL.

## Controls
- **← → or A D**: Move Left/Right
- **↑ ↓ or W S**: Speed Up/Down  
- **SPACE**: Shoot
- **P**: Pause

## Features
- Procedurally generated infinite river
- Helicopters, ships, fuel depots, bridges
- Web Audio API sound effects
- High score saved locally
- Mobile responsive

Built with HTML5 Canvas, CSS, and vanilla JavaScript.
